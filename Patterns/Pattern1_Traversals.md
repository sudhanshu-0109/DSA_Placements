# 🧠 Important Observations

- Every array problem starts with proper traversal.
- Matrix problems are mostly about managing row and column indexes.
- Spiral Matrix introduces boundary pointers (`top`, `bottom`, `left`, `right`).
- Boundary Traversal teaches careful edge handling.
- Diagonal Traversal develops confidence with index relationships.
- Wave and Zig-Zag Traversals improve matrix thinking.
