# 📚 Study Strategy

Every module follows the same workflow:

- 📖 Theory
- 📝 Notes
- 🟢 Easy Problems
- 🟡 Medium Problems
- 🔴 Hard Problems (if required)
- 🔁 Revision
- 📒 Mistake Log
- ✅ Module Completion

---

# 🎯 Daily Study Plan (3 Hours)

### 📖 Theory
**30–45 min**

- Learn concepts
- Understand patterns
- Dry run examples

---

### 💻 Problem Solving
**1 hour 45 min**

- Solve new questions
- Think for at least **20 minutes** before looking at hints
- Focus on pattern recognition

---

### 🔁 Revision
**30–45 min**

- Re-solve previous questions
- Update notes
- Record mistakes
- Review observations
