# 💡 Learning Philosophy

I don't memorize solutions.

I learn one pattern at a time and derive the optimal solution through observation.

Every new pattern follows the same journey.

```text
Brute Force

↓

Observation

↓

Optimization

↓

Pattern Recognition

↓

Revision

↓

Mixed Practice
```

The goal is to make every new problem feel familiar by connecting it to an already learned pattern.

# 🔁 Revision Rule

A module is considered **completed** only after:

- ✅ Learning the theory
- ✅ Solving all planned problems
- ✅ Writing complete notes
- ✅ Creating final revision notes
- ✅ Revising for at least two days
- ✅ Blind solving mixed questions
- ✅ Explaining the intuition without looking at notes

# ⏰ Daily Study Plan (3 Hours)

## 📖 Theory (30–45 min)

- Learn concepts
- Understand intuition
- Dry run examples

---

## 💻 Problem Solving (1 hr 45 min)

- Solve new questions
- Think for at least **20 minutes**
- Focus on recognizing patterns
- Avoid memorizing solutions

---

## 🔁 Revision (30–45 min)

- Re-solve previous questions
- Update notes
- Record mistakes
- Review observations
